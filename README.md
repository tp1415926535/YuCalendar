**介绍**
-    

**查看**   
- 万年历：公历年月日、农历月日、天干地支八字、星期、周数、当前时分、生肖、星座、星宿、节日、节气、前后节气、时辰吉凶。      
- 鱼势：宜事、天气、幸运物、幸运词。   
   
**功能**
- 关联：双击鱼势描述后，浏览器跳转至对应微博详情页。   
- 日历：前一天、后一天、回到今天、月/年历选择。   
- 窗体：展开/折叠、置顶/取消置顶、主题色变更、自定义主题色。
